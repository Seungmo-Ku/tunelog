## Tunelog (Personal Project)

- [Summary] Music rating site using Next.js and MongoDB Atlas (Reactive Design)
- [Main Features] JWT based Login/Register, Community feature, Personal Archiving, Music (singer, song, album) search and rating feature
- [Tech Stack] Next.js, Typescript, TailwindCSS, MongoDB
- [Links] https://tunelog-phi.vercel.app (website) | https://github.com/Seungmo-Ku/tunelog (Repo)
